# CodeTouch
Interview test

Steps to test the project:
1- Import the database to your localhost.
2- Download the project from github using git clone or download as zip then extract the file.
3- Copy the file "CodeTouch-master/Test/app/config/parameters.yml.dist" 
    to "CodeTouch-master/Test/app/config/parameters.yml" then open it using any text editor 
    and define the database name, user and password. Set the parameter database_port: null
4- Open terminal and browse to the project folder named "Test".
5- Run the following command "bin/console server:run".
6- Open any browser and go to this url "localhost:8000".
7- Create users as you need and proceed with testing.
